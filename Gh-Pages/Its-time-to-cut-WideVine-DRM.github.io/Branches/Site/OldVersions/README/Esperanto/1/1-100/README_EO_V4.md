## Manko de libereco

WideVine estas uzata por malebligi uzantojn interagi kun filmetoj en retejoj. ƒúi estas formo de mastrumado de ciferecaj limigoj, kiu malebligas vin el≈ùuti la filmeton, vidi la filmeton eksterrete a≈≠ eƒâ ekrankopii. ƒúi estas proprieta programaro kaj pro siaj problemoj kun privateco, ƒùi ne estas instalita defa≈≠lte ƒâe plej multaj Linuksaj distribuoj. ƒúi limigas la liberecojn de la retejo pro ƒùia uzo de Netflix, Disney kaj YouTube-filmoj. Via aliro al la enhavo povas esti forigita iam ajn sen kialo.

***

## Memora uzado

WideVine malbone memoras. Kompare kun nur kutime vidi filmeton sen DRM, WideVine uzos pezajn kvantojn de CPU kaj RAM. Estas malbone pri bamulta vivo, kaj ƒùi donas neniujn avantaƒùojn de norma HTML5-videoludado.
